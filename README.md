# دكاكين — دليل التشغيل والرفع على Firebase (جوجل)

المشروع ده نسخة كاملة من موقع "دكاكين" تشتغل برا شات كلود، وبتحفظ
البيانات (المنتجات، المستخدمين، المزايدات) على **Firebase Firestore**
بدل نظام التخزين الخاص بكلود.

---

## 1. اعمل مشروع Firebase (مجاني)

1. روح على https://console.firebase.google.com وسجل دخول بإيميلك.
2. دوس **Add project** واختار اسم (مثلاً `dakakeen`).
3. من صفحة المشروع، دوس على أيقونة **Web** (`</>`) عشان تضيف تطبيق ويب.
4. Firebase هيديك كائن (object) اسمه `firebaseConfig` فيه apiKey و
   projectId وباقي البيانات — انسخه.
5. افتح ملف `src/firebase.js` في المشروع ده، واستبدل القيم اللي فيها
   `REPLACE_ME` بالقيم اللي نسختها.

## 2. فعّل Firestore

1. من قائمة Firebase الجانبية، افتح **Firestore Database**.
2. دوس **Create database** واختار **Start in production mode**.
3. اختار أي منطقة قريبة منك.

القواعد (rules) اللي في ملف `firestore.rules` مضبوطة تسمح بالقراءة
والكتابة لأي حد — ده مناسب للتجربة بس. لو الموقع هيشتغل فعلي وفيه ناس
حقيقيين، لازم تظبط القواعد دي أحسن (تحقق من هوية المستخدم عن طريق
Firebase Authentication بدل ما تسيبها مفتوحة).

## 3. شغّل المشروع على جهازك

محتاج يكون عندك **Node.js** مثبت (نسخة 18 أو أحدث).

```bash
npm install
npm run dev
```

هيديك رابط زي `http://localhost:5173` — افتحه في المتصفح وجرب الموقع.

## 4. ارفعه على Firebase Hosting

```bash
npm install -g firebase-tools
firebase login
firebase init hosting
```

وقت الـ `firebase init`:
- اختار **Use an existing project** واختار مشروع `dakakeen` اللي عملته.
- لما يسألك عن الـ public directory، اكتب `dist`.
- لما يسألك "Configure as a single-page app؟" جاوب **Yes**.
- لو سألك يكتب فوق ملفات موجودة، جاوب **No** (الملفات دي موجودة بالفعل).

بعد كده:

```bash
npm run build
firebase deploy
```

هيديك في الآخر رابط زي:
`https://dakakeen.web.app`

ده الرابط النهائي اللي تقدر تبعته لأي حد.

## 3. ملاحظات مهمة

- **تسجيل الدخول** في الموقع ده بسيط (اسم + إيميل بس، من غير باسورد
  حقيقي) — كويس كتجربة أولى، لكن مش نظام حماية حقيقي. لو محتاج حماية
  فعلية، استخدم **Firebase Authentication** (فيه تسجيل بالإيميل
  والباسورد جاهز ومجاني).
- **إيميل الأدمن** لسه مكتوب صريح في كود `src/App.jsx` (ابحث عن
  `ADMIN_EMAIL`) — أي حد يشوف الكود يقدر يشوفه. غيّره لإيميلك لو مختلف.
- **صور المنتجات** لسه بتتحط كرابط مباشر (زي Imgur)، رفع صور من الجهاز
  محتاج خدمة تخزين ملفات زي **Firebase Storage** (خطوة إضافية لو
  حبيت تضيفها بعدين).
